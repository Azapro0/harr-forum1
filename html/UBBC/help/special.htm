Code Wrapping (IE Only)

The code panel allows you to wrap any section of text in iB tags. To do this, highlight a portion of text, and click any of the code buttons. This will result in the appropriate iB code being automatically placed at the beginning and end of the highlighted section.

Hot Keys (IE Only)

Each button has it's own hot key. To discover a button's hot key, hover your mouse over the button and a small tooltip will appear. Look for the square bracketed letter, this indicates the buttons hot key.

To use a hot key, hold down the appropriate modifier key key and press the button's hot key.

For example: Hot key = CTRL + B

Panel Modes (Modifer Keys)

Depending on how the board administrator has set up the code panel, you will see the panel in one of two ways.

The first way is with graphics enabled. In this mode, each button is represented by a small graphical icon, similar to the buttons found in Microsoft Office.

The second way is a non-graphical panel (used for low bandwidth boards). In this mode, each button is represented by an HTML <form> button.

The type of mode affects which modifier key is to be used. See the table below for full details.

Mode Button Modifier Example
Graphial CTRL CTRL + B
Form Base
ALT ALT + B
