Introduction & Using

The iB code panel provides a simple point and click interface to ikonboard's iB Code.

Every one of ikonboard's default code tags is supported and each is represented by a button. For a description of what each button does, simply hold your mouse over the button, and a small tooltip will appear, telling you what that particular button does.

To insert iB Code into your post, click the relevant code button and the appropriate tags will be automatically inserted into your post. For advanced usage, see the section on Special Features.
